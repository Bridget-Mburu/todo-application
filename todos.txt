4. working with a json server (todos.json)
--> installing json-server
----> npm install -g json-server

--> create your json file 
----> e.g filename.json

--> run your json-server
----> json-server --watch filename.json
----> json-server -w filename.json
----> json-server filename.json
----> json-server filename.json --port=7000
